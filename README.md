## 😖 구현 내용

✅ 링크를 입력하고, '추가하기' 버튼을 누르면 해당 모달이 생성됩니다.

✅ '폴더 추가+' 버튼을 누르면 해당 모달이 생성됩니다.

✅ 카드의 케밥을 누르면 팝업이 생성됩니다.

✅ 각 팝업의 '삭제하기' 또는 '폴더에 추가' 를 누르면 해당 모달이 생성됩니다.

✅ 폴더 탭에서 '공유' / '이름 변경' / '삭제' 를 누르면 해당 모달이 생성됩니다.

✅ 폴더 공유 모달에서 카카오톡/페이스북/링크 복사 기능을 사용할 수 있습니다.
